# Kalpitiya Tourism (Frontend)

This folder contains the new React + TypeScript + Tailwind frontend (Vite).

## Run

```bash
cd frontend
npm install
npm run dev
```

## Notes

- Images are expected under `frontend/assets/images` (served at `/images/...`).
- Booking is UI-only and shows an inline success message (no backend yet).
- Update the WhatsApp number in `frontend/src/components/WhatsAppButton/WhatsAppButton.tsx`.
- Optional API base URL: set `VITE_API_BASE_URL` in a `.env` file.
