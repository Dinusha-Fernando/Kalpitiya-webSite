# Joy_Kalpitiya

This repo originally contained a static multi-page website (`Index.html`, `Activities.html`, etc.).

## New modern frontend

A redesigned React + TypeScript + Tailwind frontend has been added under `frontend/`.

### Run

```bash
cd frontend
npm install
npm run dev
```

The original static HTML/CSS files were removed after the migration (they’re still available in git history if needed).
